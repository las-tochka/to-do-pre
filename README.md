# Проектная работа. Сервис To-do
## Описание
Проектная работа для курса Frondent разработаки Yandex.
Интерактивное приложение для управления списком задач с использованием локального хранилища браузера. Позволяет добавлять, удалять, копировать и редактировать задачи с сохранением состояния между сессиями.

https://github.com/las-tochka/to-do-pre.git

## Основные функции

1. Отображение текущих задач — при загрузке приложения автоматически загружаются сохранённые задачи

2. Добавление новых задач — через кнопку "Добавить" с текстовым полем ввода

3. Автосохранение — все изменения автоматически сохраняются в localStorage

4. Удаление задач — кнопка "Удалить" для каждой задачи

5. Копирование задач — кнопка "Копировать" для дублирования существующих задач

6. Редактирование задач — кнопка "Редактировать" для изменения текста существующих задач
